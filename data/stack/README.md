# Stack

## Algorithm name

Stack (Array-based implementation with push and pop)

## Author

Victor Manuel Montini (2026-03-22)

## Algorithm description

A stack is a linear data structure that follows the Last In First Out (LIFO) principle. Elements are added and removed from the same end, called the top. This implementation uses a fixed-size array as the underlying storage. Core operations are push (add element to top), pop (remove and return top element), and peek (return top element without removing). Overflow and underflow conditions are handled explicitly.
