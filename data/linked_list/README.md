# Linked List

## Algorithm name

Singly Linked List (insert at end, delete by value)

## Author

Victor Manuel Montini (2026-03-23)

## Algorithm description

A singly linked list is a linear data structure where each element (node) contains a value and a reference (pointer) to the next node. Unlike arrays, linked lists do not require contiguous memory. This implementation supports inserting a node at the end of the list and deleting a node by its value. Traversal is always done from the head node forward.
