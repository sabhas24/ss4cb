## Algorithm name

## Author

## Algorithm description

Depth-First Search is a graph traversal algorithm that explores as far as possible along each branch before backtracking. It uses recursion (or an explicit stack) to keep track of the path being explored. DFS is useful for topological sorting, cycle detection, and solving maze or puzzle problems. Time complexity is O(V + E) where V is vertices and E is edges.
